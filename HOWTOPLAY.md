# ゲーム開始ガイド

すげぇトークンクリッカーを遊ぶ方法

## 🚀 クイックスタート

1. **ファイルを開く**
   - `index.html` をブラウザにドラッグ&ドロップ
   - または、`index.html` を右クリック → 「プログラムから開く」 → ブラウザを選択

2. **ゲーム開始**
   - 画面が表示されたら、「すげぇ！クリック」ボタンを何度もクリック
   - トークンが増え始めます！

## 🎮 基本操作

### マウス・タッチ操作
- **クリックボタン**: 左クリック（タッチデバイスではタップ）
- **購入ボタン**: 左クリック（タッチデバイスではタップ）
- **タブ切り替え**: タブボタンをクリック

### キーボード操作
- 現在のバージョンではキーボード操作はありません

## 📱 対応デバイス

- ✅ Windows PC
- ✅ Mac
- ✅ タブレット
- ✅ スマートフォン

## ⚙️ 設定

### ブラウザ設定
- JavaScriptを有効にしてください（通常はデフォルトで有効）
- ポップアップブロックを無効にしてください（通知表示の場合）

### 推奨ブラウザ
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

## 💾 セーブ機能

✅ **自動セーブ**: ゲーム進行状況は5秒ごとに自動保存されます

✅ **データ永続化**: ブラウザキャッシュをクリアするまでデータが保存されます

⚠️ **注意**: 以下の場合、ゲームデータがリセットされます
- ブラウザの閲覧履歴・キャッシュをクリア
- ブラウザのローカルストレージをクリア
- シークレットウィンドウを使用した場合

## 🐛 トラブルシューティング

### Q: ゲームが真っ黒で何も表示されない
**A**: 
- ページを再読み込み（F5キー）してください
- ブラウザのJavaScriptコンソールを確認（F12 → Console）
- JavaScriptが有効になっているか確認

### Q: クリックしても数字が増えない
**A**:
- JavaScriptが有効になっているか確認してください
- ブラウザのコンソールでエラーがないか確認してください
- 別のブラウザで試してみてください

### Q: ボタンが反応しない
**A**:
- 十分なトークンを持っているか確認
- ページを再読み込みしてください
- ブラウザのキャッシュをクリア（Ctrl+Shift+Delete）

### Q: 前のセッションのデータが残っている
**A**:
- これは仕様です。続きからプレイできます
- リセットしたい場合はブラウザ開発者ツール → Application → Local Storage → 当該サイトを削除

## 🎯 ゲー

ムバランス

ゲームは以下のバランスを考慮して設計されています：

- **初期段階（0-100トークン）**: クリッカーの快感を感じる
- **成長段階（100-10,000トークン）**: 施設で自動化を進める
- **投資段階（10,000トークン+）**: ビットコインや株で資産を増やす
- **エンドゲーム（100,000トークン+）**: 複合戦略で最大効率化

## 💡 上級者向けのコツ

1. **効率的な施設購入順序**
   - コスト効率（毎秒生成 ÷ コスト）を計算して購入

2. **暗号資産の売買**
   - 価格が上がったら売却、下がったら買い増し

3. **複利の力**
   - 施設からの生産 + 投資収益で加速的に成長

4. **放置戦略**
   - 一度施設を揃えたら、放置でも増殖する

## 📊 統計情報の見方

| 統計項目 | 説明 |
|---------|------|
| **すげぇトークン** | 現在所有しているトークン数 |
| **毎秒獲得** | 1秒間に生成されるトークン数 |
| **総獲得トークン** | ゲーム開始以来獲得した合計トークン |
| **資産ポートフォリオ** | ビットコインと株の現在価値 |
| **起業家レベル** | 獲得トークンに基づいたレベル |
| **購入回数** | 施設やアップグレード購入の回数 |

## 🔄 ゲームリセット方法

ゲームを完全にリセットしたい場合：

### Chrome/Edge/Firefox
1. F12 を押して「開発者ツール」を開く
2. 「Application」タブを選択
3. 左側「Local Storage」を展開
4. 「http://...」（このサイト）を右クリック
5. 「削除」を選択
6. ページを再読み込み（F5）

### Safari
1. 環境設定 → プライバシー
2. 「Webサイトデータを管理」をクリック
3. このサイトを選択 → 削除

## 🎨 カスタマイズ

テーマ色や属性をカスタマイズしたい場合：

- **game.js**: ゲームバランスの調整（コストや生産量）
