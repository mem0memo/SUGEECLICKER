# すげぇトークンクリッカー

クッキークリッカー風のインクリメンタルゲーム。
某起業家をモチーフにした、コメディ路線の起業シミュレーター。

## 遊び方

`index.html` をブラウザで直接開くだけで動きます。

1. コインをクリックしてトークンを獲得
2. 施設を購入して毎秒自動生産
3. 暗号資産を売買して資産を増やす
4. 実績を解除してクリック値を強化

## 施設

| 施設 | 特徴 |
|------|------|
| 野菜農園 | 最初の施設。安くて地味 |
| 和牛農場 | 肉は権力の象徴 |
| SNS配信局 | 所持数 × クリック値 を毎秒自動稼ぎ |
| お笑いプロダクション | 笑いは最強のコンテンツ |
| ロケット工場 | 規制なんか関係ない |
| ブログポータル | 記事が金になる |
| 有料メルマガ | 月額課金で信者を囲い込め |
| メディア帝国 | 情報は21世紀の石油 |
| 夜の社交場 | 一杯飲んでちょめちょめ |
| ニューハーフ | 毎秒暗号資産を自動売買 |

## 暗号資産

| 銘柄 | 特徴 |
|------|------|
| 普通のトークン | 安定型。最初から購入可 |
| すごくないトークン | まあまあなデジタル資産 |
| すげぇトークン | 生産効率高・相場激しい。有料メルマガ×10で解放 |
| サナオトークン | なぜか稼げない謎の通貨。ベア相場80% |

取引量が増えるごとに一括取引の上限が解放されます（×1千 → ×1万 → … → ×1兆）。

## 実績

26種類。easy / normal / hard / legendary の4段階。
解除するたびにクリック値にボーナスが加算されます。

## セーブ・リセット

- **自動セーブ**: `localStorage` に定期保存
- **リセット**: F12 → Application → Local Storage → `game` を削除 → リロード

## 動作環境

- Chrome / Firefox / Safari / Edge（最新版）
- PC・スマートフォン対応

## 技術

HTML + CSS + Vanilla JS（フレームワークなし）
