# すげぇトークンクリッカー 要件定義

## Context

AIが生成したが動作しない状態で頓挫したプロジェクト。
コードの骨格は揃っているが、いくつかの根本的な欠落・バグにより動作不全。
まずプロジェクトを「最低限動く状態」に修復し、その後機能を肉付けしていく方針。

---

## ゲームコンセプト

- クッキークリッカー風インクリメンタルゲーム
- ホリエモンモチーフの起業家テーマ・コメディ路線
- 「すげぇトークン」をクリックで稼ぎ、施設・スキル・投資で自動収益を拡大する

## 機能要件

### 必須（MVP）

#### コア
- [x] `fmt(n)` 実装：数値を読みやすく整形（例: 1200 → "1.2K", 1500000 → "1.5M"）
- [x] クリックでトークン増加、上部バーに即座に反映
- [x] 1秒ごとに施設からの生産が加算され、Per/Sec と Tokens に反映
- [x] Earned（累計獲得）は単調増加で正しく表示
- [x] Portfolio = 現在トークン + 建物取得原価 + 仮想通貨 + 株式時価

#### スキル（Upgrades タブ）
- [x] minEarned を満たしたスキルが解放される
- [x] 購入すると clickValue が増加
- [x] 購入済みは一覧から消える

#### 施設（Buildings タブ）
- [x] 購入するたびにコストが上昇（× multiplier）
- [x] 所持数が増えると毎秒生産が増える
- [x] 残トークンが足りない場合はボタン無効化

#### 暗号資産（Crypto タブ）
- [x] virtualAPrice が2秒ごとにランダム変動
- [x] 購入すると virtualA++ 、保有数に応じて毎秒 0.5 生産
- [x] 価格変動率（%）を表示

#### 株式（Stocks タブ）
- [x] 株価が2秒ごとに変動
- [x] 保有株に対して2秒ごとに1%配当

#### 実績
- [x] 4種の実績（最初のクリック / 施設10個 / 100万トークン / 仮想通貨50個）
- [x] 解放時に通知を表示

#### 保存
- [x] 5秒ごとに localStorage へ自動保存
- [x] リロードで復元

### 任意（Nice to Have）

- [ ] Chart.js によるチャート表示（スキル成長 / 施設生産 / 仮想通貨価格 / 株価）
- [ ] ツールチップ（各カードにホバーで説明表示）
- [x] クリックアニメーション強化（フローティング数字）
- [ ] 実績追加（起業家レベル / 特定アップグレード達成など）
- [ ] セルを売却する機能（施設・仮想通貨・株）

## 非機能要件

- 外部ライブラリなし（Chart.js は任意実装時のみ）
- ブラウザで index.html を直接開くだけで動作
- Chrome / Firefox / Edge 最新版対応
- localStorage での永続化

## 検証方法

1. `index.html` をブラウザで開く
2. 上部バーに Tokens / Per/Sec / Earned / Portfolio が数値で表示されること
3. クリックボタンを押すと Tokens と Earned が即座に増加すること
4. 数秒待つと Per/Sec に応じてトークンが自動増加すること
5. スキルを購入すると clickValue が上がり、リストから消えること
6. 施設を購入すると Per/Sec が増加すること
7. リロードしても進捗が保持されること
